# SQL Marketing Analytics Exercises

## Exercise Overview
This folder contains SQL exercises for practicing marketing performance analysis using the olist marketing and sales datasets.

## Exercise Files
1. **exercise-01-campaign-performance.sql** - Analyze marketing campaign effectiveness
2. **exercise-02-marketing-funnel.sql** - Build customer acquisition funnels
3. **exercise-03-channel-attribution.sql** - Calculate attribution metrics
4. **exercise-04-marketing-roi.sql** - Compute marketing ROI and optimization

## Getting Started
1. Connect to the database containing olist_marketing_data_set and olist_sales_data_set
2. Open each SQL file in your preferred SQL editor
3. Read the problem description and requirements
4. Write your queries to solve each challenge
5. Compare your results with expected outputs
6. Check solutions if needed (instructors only)

## Database Schema
You'll be working with these key tables:
- `olist_marketing_qualified_leads` - Marketing lead data
- `olist_closed_deals` - Converted leads information
- `olist_orders` - Order transaction data
- `olist_order_items` - Product details
- `olist_customers` - Customer information

## Exercise Format
Each exercise includes:
- **Business Context** - Real-world marketing scenario
- **Requirements** - Specific analytical questions to answer
- **Hints** - Query suggestions when needed
- **Expected Results** - Sample output format
- **Challenge Questions** - Advanced variations

## SQL Techniques Practiced
- Complex JOINs across marketing and sales data
- Window functions for time-series analysis
- CTEs for multi-step calculations
- Aggregate functions for campaign metrics
- Subqueries for advanced filtering

**Time Allotment**: 30-45 minutes per exercise
**Difficulty**: Intermediate to Advanced
