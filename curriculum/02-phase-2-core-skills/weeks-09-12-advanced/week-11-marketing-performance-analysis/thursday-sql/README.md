# Thursday: SQL Marketing Performance Analysis

## Session Overview
This 2-hour session focuses on SQL-based marketing analytics using the olist marketing and sales datasets. Students will learn to query marketing data, analyze campaign performance, and derive actionable insights.

## Learning Outcomes
- Write complex SQL queries for marketing analysis
- Perform marketing funnel analysis
- Calculate marketing attribution and ROI
- Create channel performance reports
- Analyze customer acquisition patterns

## Materials Structure
```
thursday-sql/
├── exercises/           # SQL practice exercises
├── lecture-materials/
│   ├── notebooks/     # SQL query examples
│   └── scripts/       # Complete analysis scripts
├── resources/         # SQL reference materials
└── solutions/         # Exercise solutions (instructors only)
```

## Session Timeline
1. **Marketing Data Querying** (30 min)
2. **Funnel Analysis Techniques** (45 min)
3. **Channel Attribution Modeling** (45 min)
4. **Advanced Marketing Analytics** (30 min)

## Key SQL Concepts
- Window functions for marketing metrics
- CTEs for complex marketing calculations
- JOIN operations across marketing and sales data
- Aggregate functions for campaign analysis
- Time-series analysis for marketing trends

## Exercises
- Exercise 1: Marketing Campaign Performance
- Exercise 2: Customer Acquisition Funnel
- Exercise 3: Channel Attribution Analysis
- Exercise 4: Marketing ROI Calculation
