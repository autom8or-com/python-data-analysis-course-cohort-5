# Python Marketing Analytics Exercises

## Exercise Overview
This folder contains hands-on exercises for practicing Python marketing analytics techniques using the olist e-commerce dataset.

## Exercise Files
1. **exercise-01-marketing-metrics.ipynb** - Calculate basic marketing KPIs
2. **exercise-02-customer-segmentation.ipynb** - Perform RFM analysis
3. **exercise-03-ab-testing.ipynb** - Analyze A/B test results
4. **exercise-04-campaign-dashboard.ipynb** - Create marketing dashboard

## Getting Started
1. Clone the datasets from `../lecture-materials/datasets/`
2. Open each exercise in Jupyter Notebook or Google Colab
3. Follow the instructions and complete the coding tasks
4. Check your work against the solutions (instructors only)

## Required Data
- `olist_marketing_qualified_leads.csv`
- `olist_closed_deals.csv`
- `olist_orders_dataset.csv`
- `olist_order_items_dataset.csv`
- `olist_customers_dataset.csv`

## Exercise Format
Each exercise follows this structure:
- **Learning Objectives** - What you'll accomplish
- **Dataset Description** - Data you'll be working with
- **Step-by-Step Tasks** - Guided coding challenges
- **Extension Questions** - Bonus challenges
- **Expected Output** - Sample results to validate your work

---

**Time Allotment**: 30-45 minutes per exercise
**Difficulty**: Intermediate to Advanced
