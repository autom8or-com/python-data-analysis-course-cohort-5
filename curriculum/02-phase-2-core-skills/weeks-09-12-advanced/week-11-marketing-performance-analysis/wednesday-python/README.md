# Wednesday: Python Marketing Performance Analysis

## Session Overview
This 2-hour session focuses on using Python for marketing analytics, including customer segmentation, campaign analysis, and marketing ROI calculations using the olist e-commerce dataset.

## Learning Outcomes
- Calculate marketing KPIs using pandas
- Perform customer segmentation analysis
- Create marketing dashboards with visualization libraries
- Conduct A/B testing analysis
- Analyze campaign performance trends
