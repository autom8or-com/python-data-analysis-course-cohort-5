# Week 12: Financial Analysis and Reporting

## Overview
Week 12 focuses on comprehensive financial analysis and statistical reporting using SQL and Python. Students will learn to perform statistical analysis on financial data, understand correlations and relationships between business metrics, and create sophisticated business intelligence queries for executive reporting.

## Learning Objectives

### SQL (Thursday)
- Master statistical functions in SQL (STDDEV, VARIANCE, percentiles)
- Perform advanced analytical calculations with window functions
- Create sophisticated business metrics and KPIs
- Implement cohort analysis and retention rate calculations
- Build comprehensive business intelligence queries

### Python (Wednesday)
- Perform descriptive statistical analysis on financial data
- Understand correlation and relationships between business metrics
- Apply basic statistical testing for business decision-making
- Conduct A/B test analysis for financial strategies
- Create comprehensive statistical reports for executive insights

## Topics Covered

### Wednesday - Python: Statistical Analysis with Python
1. **Descriptive Statistics** (30 mins)
   - Central tendency and spread measures
   - Distribution analysis and percentiles
   - Outlier detection methods
   - Financial performance statistical summary

2. **Correlation Analysis** (45 mins)
   - Pearson and Spearman correlations
   - Correlation matrices and heatmaps
   - Understanding causation vs correlation
   - Revenue driver correlation analysis

3. **Basic Statistical Testing** (45 mins)
   - Hypothesis testing concepts
   - t-tests for mean comparisons
   - Chi-square tests for categorical data
   - A/B test analysis for marketing campaigns

### Thursday - SQL: Advanced Analytics Functions
1. **Statistical Functions in SQL** (30 mins)
   - STDDEV, VARIANCE for spread measures
   - PERCENTILE_CONT and PERCENTILE_DISC
   - Statistical aggregations
   - Customer behavior statistical analysis

2. **Advanced Window Functions** (45 mins)
   - FIRST_VALUE and LAST_VALUE
   - Complex window frame definitions
   - Analytical functions for business metrics
   - Customer journey analysis

3. **Business Intelligence Patterns** (45 mins)
   - Cohort analysis techniques
   - Retention rate calculations
   - Complex KPI definitions
   - Customer retention metrics

## Prerequisites
- Completion of Week 11: Marketing Performance Analysis
- Understanding of SQL window functions and aggregations
- Intermediate Python data manipulation skills
- Familiarity with pandas, numpy, and statistical concepts
- Basic knowledge of correlation and statistical testing

## Schedule
- **Wednesday**: Python Statistical Analysis (2 hours)
- **Thursday**: SQL Advanced Analytics Functions (2 hours)

## Business Context: Nigerian E-commerce Financial Analysis

This week focuses on analyzing the financial performance of a Nigerian e-commerce marketplace using the Olist dataset. Students will:

- Analyze revenue patterns across different customer segments and regions
- Examine payment method preferences and their impact on business metrics
- Evaluate seasonal trends and their effect on financial performance
- Conduct statistical analysis to identify key business drivers
- Build comprehensive financial dashboards for executive decision-making

## Key Datasets Used

### Olist Sales Dataset
- `olist_orders_dataset` - Order information and timestamps
- `olist_order_payments_dataset` - Payment details and methods
- `olist_customers_dataset` - Customer geographic and demographic data
- `olist_order_items_dataset` - Product and pricing information

### Olist Marketing Dataset
- `olist_marketing_qualified_leads_dataset` - Lead generation data
- `olist_closed_deals_dataset` - Conversion and revenue data
